# EvikoAppGrok - Android приложение для магазина с 3D визуализацией

## Описание
Мобильное приложение для платформы Android, представляющее собой современный интернет-магазин с возможностью просмотра 3D моделей товаров. Приложение разработано с использованием последних технологий и архитектурных подходов, обеспечивая высокую производительность, масштабируемость и удобство использования.

## Технические характеристики

### Системные требования
- Android 7.0 (API 24) и выше
- Оперативная память: минимум 2 GB
- Поддержка OpenGL ES 3.0
- Место на устройстве: ~50 MB

### Архитектура
- Clean Architecture (многослойная архитектура)
- MVVM (Model-View-ViewModel)
- Single Activity
- Compose Navigation
- Repository Pattern
- Dependency Injection с Hilt

### Используемые технологии
- **UI**: Jetpack Compose 2024.02.00
- **Навигация**: Navigation Compose 2.7.7
- **Dependency Injection**: Hilt 2.50
- **База данных**: Room 2.6.1
- **Асинхронность**: Coroutines & Flow 1.7.3
- **3D рендеринг**: Filament 1.45.0
- **Загрузка изображений**: Coil 2.5.0
- **Хранение настроек**: DataStore 1.0.0
- **Безопасность**: Security Crypto 1.1.0-alpha06

## Установка и запуск

### Требования к окружению
- Android Studio Hedgehog или новее
- Android SDK 34
- JDK 17
- Gradle 8.2.0

### Шаги установки
1. Клонируйте репозиторий:
```bash
git clone https://github.com/Positive1a17/EvikoAppGrok.git
```

2. Откройте проект в Android Studio

3. Синхронизируйте Gradle и соберите проект

4. Запустите на эмуляторе или реальном устройстве

## Структура проекта
```
app/
├── src/
│   ├── main/
│   │   ├── java/com/example/shopapp/
│   │   │   ├── data/           # Слой данных (модели, репозитории)
│   │   │   ├── di/            # Dependency Injection модули
│   │   │   ├── ui/            # UI компоненты и ViewModels
│   │   │   └── utils/         # Утилиты и расширения
│   │   └── res/               # Ресурсы приложения
│   ├── test/                  # Unit тесты
│   └── androidTest/           # Инструментальные тесты
```

## Разработка

### Сборка проекта
- Debug сборка: `./gradlew assembleDebug`
- Release сборка: `./gradlew assembleRelease`

### Запуск тестов
- Unit тесты: `./gradlew test`
- Инструментальные тесты: `./gradlew connectedAndroidTest`

### Проверка качества кода
- Lint проверка: `./gradlew lint`
- Detekt анализ: `./gradlew detekt`
